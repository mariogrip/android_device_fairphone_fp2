Fairphone FP2
==============

Fairphone FP2 is the first modular smartphone to be available for purchase.

FP2 was first released in December 2015.

Basic   | Spec Sheet
-------:|:-------------------------
CPU     | 2.26GHz Qualcomm MSM8974AB-AB (Snapdragon 800)
GPU     | Adreno 330 @ 578Mhz
Memory  | 2GB LPDDR3
Shipped Android Version | 5.1
Storage | 32GB
Battery | 2420 mAh
Display | 5"  1080×1920 px
Rear Camera  | 8MPx, LED Flash
Front Camera  | 2MP
